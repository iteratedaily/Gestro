# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x.x   | ✅ Active support |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability in Gestro, please report it by:

1. Opening a **private** GitHub Security Advisory:
   Go to → Security tab → Report a vulnerability

2. Include the following in your report:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

## What to Expect

- You will receive a response within **48 hours**
- If the vulnerability is confirmed, a fix will be prioritized
- You will be credited in the release notes (unless you prefer anonymity)

## Scope

Gestro is a Chrome extension that:
- Accesses your camera (only when toggled ON by you)
- Runs all processing locally — nothing is sent to any server
- Does not collect, store, or transmit any personal data

**All gesture detection happens 100% on your device.**
